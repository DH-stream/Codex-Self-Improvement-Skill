# Active user UX and visual taste

This is a neutral installation template. The active copy lives under `PRIVATE_LOCATION` and is never committed or pushed.

Load only for relevant visual, UX, copy, interaction, or image tasks.

## Record format

```yaml
id: taste-NNN
preference: concise durable preference
scope: universal-to-user | product-type | project | medium
status: provisional | confirmed | low-relevance | superseded
confidence: low | medium | high
source: explicit-correction | explicit-preference | repeated-choice | inference
first_observed: YYYY-MM-DD
last_reviewed: YYYY-MM-DD
```

Required sections:

```text
Apply when
Do not overgeneralize
Evidence
Conflicts
Relevance history
```

No learned preferences are stored in the public template.
